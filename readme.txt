The Visitor Manager Web Appication

Steps to install 'Visitor Manager' on your system:
	1. Extract visitor_manager.zip file in any directory on your system.
	2. Check whether your system has all the required plugins and packages installed, as per the requirements.txt file.
	3. Open the application and start a local host server.
	4. Enjoy!

Note:	To receive SMS notifications, the host is requested to contact the developer for contact number authentication.
	Otherwise, the application is likely to crash on visitor check-in/check-out.
